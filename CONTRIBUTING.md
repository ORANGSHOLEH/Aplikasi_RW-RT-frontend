# Panduan Kontribusi

1.  Selalu buat _issue_ terlebih dahulu sebelum membuat _pull request_.
2.  Gunakan format commit _Conventional Commits_.
3.  Pastikan semua tes berjalan dengan baik sebelum _push_.
